# GenAI-DeepLearning-LDA-ENA
LDA analysis code and visualization results for GenAI-supported deep learning study.
本仓库提供论文《GenAI 对学生深度学习的影响：基于认知网络和主题模型的分析》中部分分析材料，主要包括 LDA 主题模型分析代码和 LDA 可视化结果文件。

## 文件说明

- `LDA_code/`：LDA 主题模型分析代码，包括文本预处理、主题建模和可视化结果生成等步骤。
- `LDA_results/`：LDA 主题模型分析生成的 HTML 可视化结果文件，包括前期、中期和后期三个阶段的主题模型结果。

## 数据说明

由于本研究涉及高中学生在项目式学习过程中与生成式人工智能工具的人机对话数据，原始对话文本包含学生学习过程、项目方案细节及潜在隐私信息。因此，本仓库不公开原始人机对话数据。

本仓库仅提供 LDA 分析代码及 LDA 主题模型可视化结果文件，用于说明论文中主题模型分析的基本过程与结果。

## ENA 分析说明

本研究中的认知网络分析采用 ENA Web 在线平台完成。由于该平台未生成可供本地运行的源代码或可导出的 HTML 文件，因此本仓库不提供 ENA 相关代码或 HTML 文件。

ENA 分析的主要结果，包括整体认知网络图、质心轨迹图、阶段性认知网络图、叠减图及认知网络连线系数表，已在论文正文中呈现。
